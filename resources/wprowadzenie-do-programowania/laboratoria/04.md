**Lista w języku Python to jedna z najważniejszych i najczęściej używanych struktur danych. Jest to kolekcja elementów, która pozwala przechowywać wiele wartości w jednym obiekcie. Elementy w liście mogą być różnego rodzaju, w tym liczby, ciągi znaków (łańcuchy), inne listy, a nawet inne obiekty. Oto kilka cech list:**

1. **Uporządkowana: Elementy listy są przechowywane w określonej kolejności, co oznacza, że można odwoływać się do nich poprzez indeksy.**
2. **Zmienna długość: Listy mogą zmieniać swój rozmiar, co oznacza, że można dodawać lub usuwać elementy w trakcie działania programu.**
3. **Różnorodność elementów: W jednej liście można przechowywać elementy różnych typów danych.**
4. **Indeksowanie: Można uzyskiwać dostęp do elementów listy poprzez ich indeksy, zaczynając od 0.**

**Przykład:**

**moja_lista = \[1, 2, 3, 4, 5\]**

**print(moja_lista\[0\])  # Wyświetla pierwszy element listy (1)**

**print(moja_lista\[2\])  # Wyświetla trzeci element listy (3)**

**# Można również zmieniać elementy listy**

**moja_lista\[1\] = 10**

**print(moja_lista)  # Wyświetla listę po zmianie \[1, 10, 3, 4, 5\]**

**Zadanie 1: Obliczanie średniej z listy liczb.**

Napisz program, który oblicza średnią liczb z danej listy.

```
def srednia(lista): if len(lista) == 0: return 0 suma = sum(lista) srednia = suma / len(lista) return srednia lista = [1, 2, 3, 4, 5] print("Średnia:", srednia(lista))
```

**Zadanie 2: Znajdowanie największej i najmniejszej wartości w liście.**

Napisz program, który znajduje największą i najmniejszą wartość w danej liście.

```
def min_max(lista): if not lista: return None, None min_val = min(lista) max_val = max(lista) return min_val, max_val lista = [10, 5, 15, 3, 20] min_val, max_val = min_max(lista) print("Najmniejsza wartość:", min_val) print("Największa wartość:", max_val)
```

**Zadanie 3: Usuwanie duplikatów z listy.**

Napisz program, który usuwa duplikaty z danej listy.

```
def usun_duplikaty(lista): return list(set(lista)) lista = [1, 2, 2, 3, 4, 4, 5] lista = usun_duplikaty(lista) print("Lista po usunięciu duplikatów:", lista)
```

```
lista1 = [1, 2, 3] lista2 = [4, 5, 6] polaczona_lista = lista1 + lista2 print("Połączona lista:", polaczona_lista)
```

**Zadanie 5: Odwracanie listy.**

Napisz program, który odwraca kolejność elementów w danej liście.

```
def odwroc_liste(lista): return lista[::-1] lista = [1, 2, 3, 4, 5] odwrocona_lista = odwroc_liste(lista) print("Odwrócona lista:", odwrocona_lista)
```
